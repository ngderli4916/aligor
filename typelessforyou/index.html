English

Before you start

Typeless has a free weekly word allowance. You can use up to 8,000 words per week, and the allowance resets every Monday.

If you install Typeless on both mobile and computer, do not use the same email on both devices if you want separate weekly word limits. Use one email for mobile and another email for computer.

中文

开始使用前

Typeless 有免费的每周字数额度。你每个星期可以免费使用 8,000 字，额度会在每个星期一重置。

如果你同时在手机和电脑安装 Typeless，想要分开计算每周字数额度，就不要在两个设备使用同一个邮箱。手机用一个邮箱，电脑用另一个邮箱。
