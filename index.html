Why this page exists

我做生意的方式很简单：
真实卖货，真实服务，真实自动化。

我不喜欢空谈。一个系统如果不能让我少问一次库存、少追一次物流、少复制一次客服回复、少漏一张订单，它就还不算完成。

我相信 SME 老板不缺努力。
缺的是一套不会累、不会忘、不会情绪化的第二大脑。

所以我一直把自己的公司当成实验场：BigSeller 日销报告、Shopee / Lazada 客服流程、中国到马来西亚的运输追踪、Meta 内容发布、网站搭建、团队通知。凡是重复、容易漏、需要人盯的动作，都应该先变成 SOP，再交给 AI Agent 执行。

我自己用过 Openclaw 小龙虾、Hermes Agent、Codex、ChatGPT、Claude、Gemini 等工具。不同工具有不同用途；如果要让中小型企业真正落地，我会把重点放在 AI Agent on Openclaw 小龙虾，因为它适合做自动回复、顾客沟通整理，以及后台 program 的持续运作。

我也常用 Codex 做自己的工具：改网站、写脚本、整理资料、测试自动化流程。很多人花大钱测试工具和系统；但对中小型企业来说，更实际的是先找到便宜、好用，甚至免费的方式，让流程先跑起来。

这不是为了看起来很科技。这是为了让一门小生意，也可以拥有大公司的纪律、速度和可复制性。

The Reframe

大多数 SME 老板以为自己有 AI 问题。
其实没有。

真正的问题通常是这些：

01

营运问题：资料散在 WhatsApp、Excel、平台后台和脑袋里。

02

跟进问题：客户问了，货到了，款付了，但没有一个稳定提醒系统。

03

内容问题：每天都知道要发帖，但老板太忙，Facebook、Instagram、TikTok、Threads 最后都没有稳定更新。AI Agent 可以帮你安排内容，并定时发布到社交平台。

04

老板瓶颈：每个决定、每个检查、每个例外，都要回到老板身上。

05

误解问题：很多人以为要会写 program 才能做自动化。其实现在已经进入 Vibe Coding：你把目标、场景、流程和感觉讲清楚，AI 就能帮你生成 app 或后台 program。

Founder story
A

我不是靠烧钱开始。
我是靠工程背景、资料搜寻、自己动手开始。

Ukunili / Nili Guitar 的核心很朴素：让更多人买到合适的乐器，学会音乐，继续玩下去。可是当订单、平台、供应链、客服、内容和团队一起成长，问题就不再只是“卖更多”。

真正困难的是：怎样让每一天都不乱？怎样让员工不用一直问老板？怎样让客户等少一点？怎样让内容稳定产出？怎样知道昨天到底卖了什么、哪一个 SKU 正在动、哪一个环节卡住？

我本身是 Computer Engineering 和 Software Engineering 出身，所以在写 program、找资料、测试工具这方面，我不会第一时间花大钱请人做。我会先找便宜、开源、免费，或者可以自己拼起来的方法。

但这不代表只有会写 program 的人才适合学。现在已经是 Vibe Coding 的时代：你把想要的结果、生意场景和流程讲清楚，AI 就能帮你生成 app，或者写出在后台运作的 program。重点不是你会不会写代码，而是你能不能把问题讲清楚。

我开始用 AI，不是因为它热门。是因为我受够了重复劳动，也看到很多中小型企业其实不需要一开始就买很贵的系统。

Operator first. 不是先烧钱，也不是先买工具；先用最低成本把自己的流程跑顺。

我现在更关心的是：怎样把 SME 每天遇到的碎片工作，变成可追踪、可复用、可交给团队的系统。客服回复、网站搭建、运输追踪、内容生产、内部报表，都有低成本甚至免费的做法。

所以我想教的不是“买最贵的软件”，而是怎样把 Openclaw 小龙虾、Codex、Hermes Agent、ChatGPT、Claude、Gemini 放进真实场景，用最小成本让流程自动运作。

目前我自己已经有三个网站在运行，所以这些不是理论，而是从真实生意里一边用、一边修出来的方式。

如果一个自动化不能让真实工作变轻，它就只是玩具。

Vibe Coding

把你想要的功能、场景、流程和感觉讲清楚，AI 就可以帮你生成 app 或后台 program。表达能力比代码能力更重要。

Codex / Hermes Agent

用 Codex 和 Hermes Agent 做自己的工具

改网站、写脚本、整理资料、做自动化，不一定每次都要找 agency 或 developer。能自己测试的，就先低成本测试。

06
Social Media Scheduler

自动安排和发布社交平台内容

很多老板忙到最后没有办法统一更新社交平台。AI Agent 可以帮你整理内容、安排日期，并且定时发布到 Facebook、Instagram、TikTok 或其他平台。

07
Freight Tracking

从中国到马来西亚的运输追踪

把运输状态、到货提醒、团队通知这些动作整理成固定流程，让货在哪里、下一步做什么更清楚。

我的原则：不要为了自动化而自动化，也不要一开始就烧大钱。先找最痛、最重复、最容易出错的动作，然后用最低成本把它变成系统。

Operating principles

我做事的底层逻辑。

  • 01真实场景优先没有订单、客户、库存、团队，就没有系统设计的意义。REAL OPS
  • 02能自动就不要重复手动重复三次以上的事，就应该开始写 SOP 或做自动化。DRY
  • 03老板要从瓶颈变成架构师不是每件事都亲自做，而是设计一套别人也能执行的方式。SYSTEM
  • 04先找免费或低成本方法很多 Guru 花大钱测试，但 SME 更需要的是便宜、够用、能落地。LOW COST
  • 05不需要先会写 programVibe Coding 的重点是把需求、场景和流程讲清楚，让 AI 生成你要的工具。VIBE CODING
  • 06AI 不能代替判断AI 负责加速；老板负责方向、价值观和最后判断。JUDGMENT
