Facebook × AI Agent 小白教学

如何把 Facebook 连接到 AI Agent，让它帮你处理内容、留言和客户线索

这不是叫 AI 帮你“想文案”而已，而是让 AI Agent 连接你的 Facebook Page，帮你执行任务、整理资料、发帖、回复、追踪客户和做自动化。

一句话

连接之后可以做什么

AI Agent 可以帮老板处理重复工作：内容、留言、私信、客户资料、跟进和报告。

最简单的理解方式

  • ✓Facebook Page 是你的店。
  • ✓Meta App 是门口的钥匙系统。
  • ✓Access Token 是钥匙。
  • ✓Webhook 是门铃。
  • ✓AI Agent 是你的员工。
01 · What It Can Do

连接之后，AI Agent 可以帮 Facebook 做什么？

AI Agent 不只是聊天机器人。它可以帮老板处理重复工作：内容、留言、私信、客户资料、跟进和报告。

内容规划和写文案

每周安排发帖主题，写 Facebook post、活动介绍、产品故事、老板个人品牌内容。

自动或半自动发帖

把准备好的内容发布到 Facebook Page，也可以改成 Instagram / Threads 不同版本。

留言整理

整理客户留言、常见问题、投诉、问价钱的人，并标记哪些需要真人处理。

Messenger 自动回复

回复常见问题，发送价格表、catalog、预约链接，或引导客户去 WhatsApp。

客户线索整理

整理谁问价钱、谁要地址、谁有兴趣报名、谁需要 follow up。

社交媒体报告

总结哪些 post 表现好、哪些内容没人看、客户最常问什么、下周应该发什么。

02 · Prompt

第一步：你要先这样交代你的 AI Agent

你不需要会 programming。你只需要把目标讲清楚：你是谁、做什么生意、要 AI 帮你做什么、哪些可以自动、哪些必须问你。

我要把我的 Facebook Page 连接到 AI Agent。

我的目标：
1. 帮我写 Facebook 内容
2. 帮我自动或半自动发帖
3. 帮我整理留言和 inbox
4. 帮我自动回复简单问题
5. 帮我把潜在客户引导去 WhatsApp
6. 每个星期给我一份 Facebook 表现报告

我的 Facebook Page 名字是：____
我的行业是：____
我的产品/服务是：____
我的 WhatsApp link 是：____
我的网站是：____

请你用小白方式，一步一步带我完成。
不要叫我给你 Facebook 密码。每一步只告诉我做什么，我完成后再继续。
重点：不是你要懂 API，而是你要懂得交代 AI：我要什么结果？哪些可以自动？哪些必须先问老板？
03 · Prepare

第二步：你需要准备什么东西？

你需要有 Facebook Page

注意，是 Facebook Page，不是个人 Profile。AI Agent 通常适合连接公司 Page、品牌 Page、个人品牌 Page。

你必须是 Page Admin

打开 facebook.com/pages/?category=your_pages，确认你的 Page 出现在 “Pages you manage”。如果只是 follow 的 Page，就不能连接。

你需要一个 Meta Developer App

Meta App 是 Facebook 官方给外部系统连接 Facebook 的桥梁。可以到 developers.facebook.com 创建。

你要准备业务资料

例如产品资料、FAQ、价格表、营业时间、WhatsApp link、网站、回复语气和哪些情况要通知真人。

04 · Setup

第三步：实际连接 Facebook 的流程

  1. 打开 Meta Developer。
    去 https://developers.facebook.com/apps/，检查有没有现有 App。没有的话就 Create App。
  2. 选择 Business 类型。
    通常选 Business。不要乱选 Gaming、Consumer 之类的类型。
  3. 加入需要的产品。
    常见是 Facebook Login、Graph API、Messenger API。如果也要 Instagram，就加入 Instagram Graph API。
  4. 选择需要的权限。
    例如 pages_show_list、pages_read_engagement、pages_manage_posts、pages_messaging、pages_manage_metadata。
  5. 打开 Graph API Explorer。
    去 https://developers.facebook.com/tools/explorer/，选择你的 App，然后 Generate Access Token。
  6. 授权并复制短期 User Token。
    Facebook 会问你是否授权。授权后会出现一串很长的 token，通常以 EAA 开头。
  7. 把短期 token 换成长效 token。
    Graph API Explorer 的 token 通常很快过期。要让 AI 长期运作，需要换成 60 天 User Token。
  8. 取得 Page Token。
    用长期 User Token 取得 Page Token。之后 AI Agent 用 Page Token 操作你的 Facebook Page。
  9. 测试连接。
    先测试能不能读取 Page，再测试能不能发测试 post。真正发布前，建议先让老板确认。
  10. 如果要自动回复，继续设置 Webhook。
    Webhook 是门铃。客户一留言或私信，Facebook 就通知 AI Agent，AI 才能马上处理。
05 · Token Logic

小白必须懂：Token 就是钥匙

很多人连接失败，是因为没有分清楚短期 token、长期 token 和 Page token。

短期 User Token→60 天 User Token→Page Token→AI Agent 操作 Facebook Page
不要公开 token。Access Token 就像钥匙。别人拿到后，可能可以操作你的 Page。不要截图公开、不要放进网页、不要放 GitHub。
06 · Automation

哪些东西可以交给 AI 托管？哪些不建议？

适合 AI 托管

  • ✓FAQ 回复：价钱、地址、营业时间、课程资料、产品介绍。
  • ✓Lead capture：姓名、电话、预算、地区、想买什么。
  • ✓内容生产：post、caption、活动宣传、个人品牌文案。
  • ✓评论整理：问价钱、投诉、潜在客户、需要人工回复。
  • ✓Follow up 提醒和每周表现报告。

不要完全交给 AI

  • ×客户生气、退款、法律、恶意留言、公开争吵。
  • ×金钱决定：折扣、退款、大额交易、付款确认。
  • ×品牌危机：负评爆发、政治、宗教、种族内容。
  • ×广告预算：开广告、加预算、改 audience、改 payment。
  • ×客户隐私：电话、地址、付款证明、身份证、银行资料。
07 · Control Level

推荐从半自动开始，不要一开始全自动

等级AI 做什么真人做什么
Level 1：AI 只帮你写写 post、reply 草稿、content calendar、广告文案。检查、复制、发布。
Level 2：AI 整理和提醒整理留言、inbox、潜在客户、每天报告、follow up 提醒。亲自回复重要客户。
Level 3：AI 半自动回复回复简单问题、发送资料、引导去 WhatsApp、把复杂问题交给真人。处理成交、投诉、特殊情况。
Level 4：AI 自动化经营流程自动发帖、自动回复、分类客户、报告、连接 CRM / Sheet / WhatsApp / Email。看报告、做决策、优化策略。
08 · Safety

重要注意事项

Personal Profile 和 Page 不一样。
AI Agent 通常适合连接 Facebook Page，不适合直接自动化个人 Profile。
Instagram 个人账号功能有限。
如果要自动发布和管理，最好转成 Business 或 Creator Account。
Token 会过期。
如果没有换成长效 token，今天可以用，明天可能就坏。
WhatsApp 适合做成交和 follow up。
Facebook 可以引流，WhatsApp 可以继续沟通、成交和维护关系。
自动回复不能乱开。
先让 AI 写草稿，你确认后才发。稳定后再开放自动回复。
AI 不是代替老板。
AI 适合做重复、繁琐、整理型工作。老板负责方向、价格、关系和决策。
