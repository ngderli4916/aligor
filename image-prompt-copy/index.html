01 · 修复高清图

模糊照片修复成 4K

这个 prompt 可以修复 4K 高清图。

完整 Prompt 
把这个图片的画质修复到 4k，严禁任何形式的自动美颜、磨皮、大眼、风格化处理或面部结构改变。禁止一切光滑或塑料感皮肤质感。深入刻画自然的物理瑕疵，包括皮肤毛孔、细微皱纹、自然的肤色变异、不均匀的肤色分布以及自然的微小阴影。必须保留细微的不对称性。精细化重现微观细节。每一根发丝都要分明，衣服上的织物纤维、表面的灰尘、指纹、以及表面的真实微小反射都必须被刻画出来。
02 · 亚马逊商品图

一键生成整套商品图

这个 prompt 可以生成亚马逊商品图。

成品图位置 放白底图、场景图、卖点图拼图
完整 Prompt 
参考上传的产品图，生成一套亚马逊跨境电商商品图，包含：白底主图、配件图、桌面场景图、用法演示图、卖点海报、细节特写、操作示意图、包装展示图。保持产品和包装盒外观一致。要求商业实拍、欧美简约风、画面干净、真实写实、高级布光、细节清晰、高清、无水印、无杂乱、无变形、无乱码。
03 · 黑白线稿

照片变高级矢量线稿

这个 prompt 可以生成黑白线稿。

成品图位置 放转换后的建筑或物体线稿
完整 Prompt 
参考上传照片，将其转化为干净的黑白矢量线稿。保留原图的完整几何结构。去除所有背景、天空、色彩、光影、材质纹理和照片质感，只保留清晰精准的黑色线条。线条要有粗细层次，外轮廓稍粗，内部结构线和栏杆线更细，画面干净，白色背景，专业建筑手绘线稿风格，适合导出为 SVG / AI 矢量图。
04 · LINE Sticker

宠物照片变 LINE 表情包

这个 prompt 可以生成 LINE 表情包。

成品图位置 放 16 或 24 张贴纸效果拼图
完整 Prompt 
把上传的宠物照片做成一套 LINE Sticker 表情包。保留宠物最明显特征，做成可爱 Q 版角色，透明背景，粗描边，大表情，24 个日常聊天动作：早安、晚安、谢谢、收到、OK、不要、哈哈哈、饿了、累了、生气、委屈、比心、加油、马上来、等一下、想你、拜托、太棒了、我到了、晚点回、在忙、哭哭、惊讶、明天见。风格干净可爱，适合聊天使用。
